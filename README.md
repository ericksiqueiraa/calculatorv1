This project was developed using the following technologies:

## 🚀 Getting started

Clone the project and access the folder.

```bash
git clone https://github.com/ericksiqueiraa/calculatorv1

cd calculatorv1
```

Run this command to install the dependencies.

```bash
npx install

npx start
```
